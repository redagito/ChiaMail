Overview

ChiaMail is a Windows program that lets you send personalized mass emails through your Gmail account. You provide a list of recipients (first name, last name, email), write one email template, and ChiaMail sends a customized copy to each person — automatically filling in their name. You can add your company logo, attach files, and control how fast emails are sent to avoid getting blocked by Gmail.

What You Can Do

  • 📧 Send the same email to many people, with each person's name filled in automatically
  • 📋 Load recipient lists from an Excel-like file (.csv)
  • 🖼️ Add your company logo to every email
  • 📎 Attach files like PDFs or images (up to 25 MB total)
  • ⏱️ Set a pause between emails to stay within Gmail's limits
  • 👁️ Preview an email before sending to check it looks right
  • 📸 Take a screenshot with the 📷 button or Ctrl+Shift+S
  • 📝 A log file (ChiaMail.log) records everything that happens

Prerequisites & Setup

What You Need

  1. A Gmail address (personal or work/school)
  2. 2-Step Verification turned on for your Google account
  3. An App Password — a special 16-character code Google gives you

Getting Your App Password

  1. Go to Google's App Passwords page
  2. Sign in to your Google account if asked
  3. Under "Select app", pick Mail
  4. Under "Select device", pick Other (Custom name) and type "ChiaMail"
  5. Click Generate
  6. Copy the 16-character code that appears (it looks like xxxx xxxx xxxx xxxx)
Important: Your regular Gmail password will not work here. You must use this special App Password. If you don't see the App Password option, make sure 2-Step Verification is turned on first.

Setting Up Your Recipient List

Create a file with your recipients. The easiest way is to use a spreadsheet program like Excel or Google Sheets and save it as a ".csv" file. Include these three columns (the names don't have to be exact — ChiaMail will find them):

EmailFirstNameLastName
alice@example.comAliceJohnson
bob@example.comBobSmith

Extra columns are fine — they'll be ignored. A sample file (sample.csv) is included with the app to use as a template.

Limits: max 50 MB file size, max 10,000 recipients.

1. Gmail Account

Fill in your Gmail address and the 16-character App Password you generated.

Test Connection

Click Test Connection to make sure everything works:

  • Checked "Send test email": ChiaMail sends a real test email to your own inbox. Check that you received it.
  • Unchecked "Send test email": Only checks that ChiaMail can reach Google's email servers. No email is sent.
Tip: Start without the checkmark to quickly test your internet connection, then check it to fully confirm your App Password works.

Need an App Password? Click the Google App Passwords link in the app to go directly to the page where you can create one.

2. Recipients (CSV)

Click Browse… to pick your recipient file. The data shows up in a table and the app tells you how many people it found.

📷 Insert screenshot-csv.png — CSV loaded with preview grid showing rows

What Happens When You Load a File

  • The app finds the Email, FirstName, and LastName columns automatically (even if the names aren't exact)
  • Duplicate email addresses are skipped and reported
  • Invalid email addresses are skipped and reported
  • Blank rows are ignored
  • You'll see a summary of any issues found
If no valid recipients are found, the app will show an error and won't proceed. Make sure your file has an Email column with proper addresses like name@example.com.

3. Email Content

📷 Insert screenshot-content.png — Email Content section with subject, body, placeholder buttons

Subject

Type your email subject line. You can use {FirstName} and {LastName} to personalize it — these will be replaced with each person's actual name when sent. If you add {Logo} to the subject, it will be removed automatically (logos only appear in the body).

Body

Choose how you want to write your email:

ModeWhat It Does
Plain Text Just type normally, like a regular email. Line breaks and spacing are preserved. Best if you don't know HTML.
HTML You can use HTML codes for rich formatting — bold text, tables, colored sections, links, etc. Use this if you know HTML.

Personalization (Placeholders)

Click the buttons below the subject field to insert special tags that get replaced with each recipient's information:

TagReplaced With
{FirstName}The person's first name (from your list)
{LastName}The person's last name (from your list)
{Logo}Your company logo image (body only, not subject)
Pro tip: Place {Logo} in the body wherever you want the logo to appear. If you don't, it will be added automatically at the bottom of the email.

4. Company Logo (Optional)

Pick an image file to include your company logo in every email. Supported formats include PNG, JPG, GIF, BMP, WebP, and more.

📷 Insert screenshot-logo.png — Logo section with an image selected and preview visible
  1. Click Browse… and select your logo image
  2. A small preview of the logo appears
  3. Uncheck Include logo in emails to temporarily disable it without removing the file
  4. Place {Logo} in your email body where you want the image, or let it appear at the bottom automatically
Note: The logo is embedded directly into each email, not linked from a website. Most email programs (Gmail, Outlook, Apple Mail) show it automatically. If recipients see a broken image, their email app may not support embedded images.

5. Attachments

Attach files to every outgoing email. Click Add Files… and select one or more files. Each file appears in a list with a remove button.

Limits

  • Total attachment size cannot exceed 25 MB (Gmail's limit)
  • The check runs before any emails are sent
  • Duplicate files (same path) are silently ignored

6. Preview Email

Click Preview Email to see exactly how your email will look with placeholders replaced using the first recipient from your CSV.

📷 Insert screenshot-preview.png — Preview window showing resolved subject and body

The preview window shows:

  • From: your Gmail address
  • To: first recipient's email
  • Subject: subject with {FirstName}/{LastName} resolved
  • Body: body text with all placeholders resolved
  • Mode indicator (Plain Text / HTML)
  • Logo and attachment indicators if included
Tip: Use preview to verify your placeholder syntax before sending to hundreds of recipients. One typo in {FirstName} would be caught here.

7. Sending Emails

📷 Insert screenshot-sending.png — Bottom bar with progress bar and status log during send

Bottom Bar Controls

Delay

Set the number of seconds to wait between each email. Gmail's sending limits are approximately 500 emails/day for free accounts and 2,000/day for Google Workspace. A delay of 5–10 seconds is recommended for bulk sends. Only digits are accepted.

Send Test to Self

Sends a test email to your own address (the one you entered in the Gmail Account section). It uses whatever subject, body, logo, and attachments you've set up. This ignores the recipient list — it's just to verify everything works.

Send All Emails

Starts sending to all your recipients. Before sending, the app:

  1. Checks that everything is filled in correctly
  2. Shows a confirmation screen with a summary
  3. Tests that it can reach Google's email servers
  4. Sends to each person one at a time, with a pause between each

During Send

  • A progress bar shows completion percentage
  • The status log updates after each recipient
  • Errors per recipient are logged but don't stop the batch
  • Click Cancel to stop mid-batch (pending sends are cancelled)
  • Closing the window during send prompts for confirmation

After Send

  • The status log shows results for every recipient (✓ success / ✗ error)
  • All log entries are saved to ChiaMail.log next to the application EXE
  • The log is cleared each time the app starts

Troubleshooting

"SMTP error — Authentication failed"

  • Make sure 2-Step Verification is turned on for your Google account
  • You must use an App Password — your regular Gmail password won't work
  • App Passwords are 16 characters long (spaces don't matter)
  • If you created a new App Password, update it in the app

"Could not reach email server"

  • Check your internet connection
  • Your firewall or VPN might be blocking the connection
  • Some workplace or school networks block outgoing email. Try from a different network.

"CSV has no valid recipients"

  • Make sure your file has a column called Email (capitalization doesn't matter)
  • Check that email addresses are written correctly, like name@example.com
  • Remove any duplicate email addresses
  • Make sure the file isn't empty and has at least one row of data

"Attachment size exceeds limit"

  • Total attachments per message cannot exceed 25 MB
  • Reduce file sizes or remove large attachments
  • Consider hosting large files and linking them instead

"Logo file not supported"

  • Supported formats: PNG, JPG, GIF, BMP, WebP, SVG, ICO, TIFF
  • Rename your file to have the correct extension
  • If the file is corrupted, try re-saving it from your image editor

Taking Screenshots

Click the 📷 button in the bottom bar or press Ctrl+Shift+S on your keyboard to save a picture of the current window. You'll be asked where to save it. This is useful for documentation or sharing what you see with someone else.

Windows Defender / SmartScreen: Since this app isn't digitally signed, Windows may show a warning the first time you run it. Click "More info" then "Run anyway" to proceed. This is normal for programs that don't have a paid publisher certificate.
