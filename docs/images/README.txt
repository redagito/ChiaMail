SCREENSHOT GUIDE
================

Run ChiaMail and use the 📷 button (or press Ctrl+Shift+S) to capture
these screenshots. Save each with the exact filename listed below.

1. screenshot-main.png
   - Full app window with all sections visible (scroll to show everything)
   - CSV loaded with sample.csv, subject/body filled, logo selected

2. screenshot-account.png
   - Scroll to show only the Gmail Account section
   - Email and App Password filled in, "Send test email" checked

3. screenshot-csv.png
   - Scroll to show only the Recipients section
   - sample.csv loaded, preview grid showing rows

4. screenshot-content.png
   - Scroll to show only the Email Content section
   - Subject and body filled with {FirstName} and {LastName}

5. screenshot-logo.png
   - Scroll to show only the Company Logo section
   - Logo selected, preview visible, checkbox checked

6. screenshot-attachments.png
   - Scroll to show only the Attachments section
   - A few files added to the list

7. screenshot-preview.png
   - Click "Preview Email" button, capture the Preview window
   - Shows resolved subject/body with placeholders replaced

8. screenshot-sending.png
   - Bottom bar visible with progress bar active during a send
   - Or bottom bar with status log entries visible after a send

Save all files in this directory (docs/images/).
Then open docs/index.html in a browser to see the complete guide.
